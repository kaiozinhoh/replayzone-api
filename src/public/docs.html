🔐 Autenticação

⚠️ Importante: Todas as rotas com prefixo /api requerem uma API Key válida no header x-api-key
Header obrigatório: x-api-key: Kaioh@30052003
Origens permitidas: https://app.replayzone.com.br, https://admin.replayzone.com.br, http://localhost:5173

🌐 Rotas Públicas (Sem Autenticação)

GET /health
Health check da API - retorna status do serviço
POST /teste-inadimplencia
Executa manualmente verificação de inadimplência e bloqueio de quadras
GET /status-cliente/:clienteId
Verifica status de um cliente específico
POST /vincular-cliente-quadra
Vincula um cliente a uma quadra

👥 Administração

🔐 Autenticação de Admin

POST /api/adminlogin
Login de administrador
POST /api/admin-forgot-password
Recuperação de senha de admin
POST /api/adminregister
Criar novo administrador

⚙️ Gestão de Administradores

GET /api/admins
Listar todos os administradores
GET /api/admins/:id
Obter administrador específico
PUT /api/admins/:id
Atualizar dados completos do administrador
PATCH /api/admins/:id/cargo
Atualizar apenas o cargo do administrador
DELETE /api/admins/:id
Excluir administrador
PUT /api/admins/:id/password
Atualizar senha do administrador

👤 Usuários

🔐 Autenticação

POST /api/login
Login de usuário comum
POST /api/register
Criar novo usuário
POST /api/phone-login
Iniciar login com telefone
POST /api/verify-code
Verificar código enviado por SMS

⚙️ Gestão de Usuários

GET /api/getbyiduser/:id
Obter dados de um usuário
PUT /api/changeuser/:id
Atualizar dados do usuário
PUT /api/changepassword/:id
Alterar senha do usuário
PUT /api/changepasswordemail/:email
Alterar senha por e-mail
DELETE /api/deleteuser/:id
Excluir usuário

🏟️ Quadras

📋 Gestão de Quadras

GET /api/quadras
Listar todas as quadras
GET /api/quadrabyid/:id
Obter quadra por ID
GET /api/quadras/slug/:slug
Obter quadra por slug (URL amigável)
GET /api/quadrasuser/:usuario_id
Obter quadras de um usuário específico
POST /api/quadras
Criar nova quadra
PUT /api/quadras/:id
Atualizar quadra existente
DELETE /api/deletequadra/:id
Desvincular/excluir quadra

🔗 Vínculos e Streaming

POST /api/vincular-usuario
Vincular usuário à quadra
POST /api/quadras/:quadra_id/restart-hls
Reiniciar todos os streams HLS de uma quadra
POST /api/subquadras/:sub_quadra_id/restart-hls
Reiniciar stream de uma subquadra específica

⏰ Horários e Patrocinadores

GET /api/horario-funcionamento/:quadra_id

PUT /api/horario-funcionamento/:quadra_id
Atualizar horário de funcionamento
GET /api/sponsors/:quadra_id

PUT /api/sponsors/:quadra_id
Atualizar patrocinadores
